# Customer Segmentation Dashboard
This project presents an interactive dashboard built using the Online Retail dataset, aimed at  analyzing customer purchasing behavior and identifying meaningful customer segments. 
## Objective
The dashboard provides a comprehensive view of sales performance, customer activity, and segmentation insights. It helps uncover patterns in customer behavior, enabling better decision-making for marketing strategies and business growth.
## Dataset Used
- https://l1nk.dev/2xcq79e
## Key Features 
## Sales Analysis  
- Total revenue, number of transactions, and quantity sold  
- Time-based trends (monthly/weekly performance)  
## Customer Insights  
- Identification of high-value customers  
- Purchase frequency and spending patterns  
## Segmentation (RFM Analysis)  
- Customers grouped based on:  
- Recency – how recently they purchased  
- Frequency – how often they purchase  
- Monetary Value – how much they spend  
- Segments such as:  
- Loyal customers  
- At-risk customers  
- High-value customers  
## Geographical Analysis  
- Sales distribution across different countries/regions
##Interactive Filters  
- Dynamic filtering by date, region, and customer segments
